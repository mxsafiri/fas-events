# Event Photos

Add your event photos here with these filenames:

- `event1.jpg` - Wedding/Corporate event
- `event2.jpg` - Birthday/Private event  
- `event3.jpg` - Catering showcase
- `event4.jpg` - Décor setup
- `event5.jpg` - Full event view

## Image Specifications

- **Size**: 1920x1080px (landscape orientation)
- **Format**: JPG or PNG
- **Quality**: High resolution but optimized (under 500KB each)
- **Content**: Professional photos showcasing your best work

## Tips

- Use bright, well-lit photos
- Show variety (weddings, corporate, birthdays, etc.)
- Include close-ups of décor and food
- Feature happy clients when possible
- Ensure photos are properly licensed/owned by you
